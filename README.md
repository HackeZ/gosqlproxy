# GoSQLProxy

GoSQLProxy is a simply proxy of MySQL route write/read request to master/slaves instances.

## Getting started

### Getting GoSQLProxy

running go get command download package to `$GOPATH`.

`$ go get github.com/hackez/gosqlproxy`
